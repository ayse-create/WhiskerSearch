# WhiskerSearch - ProfessorCat Web Sitesi

Bu klasör, WhiskerSearch-ProfCat projesinin web sitesini içerir.

## Dosyalar

- **index.html** - Ana HTML dosyası
- **styles.css** - Tüm stil tanımlamaları
- **script.js** - JavaScript fonksiyonları ve interaktif öğeler
- **README.md** - Bu dosya

## Özellikler

🐱 **Tarayıcı Simülasyonu** - Chrome benzeri arayüz
🔍 **Arama Fonksiyonelliği** - ProfessorCat'ta arama yapın
🤖 **CatAI** - Yapay zeka asistanı (Web-LLM tabanlı)
🎨 **Modern Tasarım** - Gradients ve animasyonlar
📱 **Responsive** - Tüm cihazlarda uyumlu

## Sekmeleri

- Tümü - Arama sonuçları
- Görseller - Kedi emojileri
- Videolar - Video öğeleri
- Alışveriş - Ürün listesi
- AI Modu - Chatbot

## Kurulum

Browser'de index.html'yi açıp kullanabilirsiniz.
